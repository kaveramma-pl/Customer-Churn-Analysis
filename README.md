# Telecom Customer Churn Analysis

## Project Overview
This project analyzes telecom customer churn patterns using Python, SQL, and Power BI.

The objective was to identify:
- key churn drivers
- high-risk customer segments
- customer behavior trends

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI

## Key Insights
- Month-to-month customers showed the highest churn.
- Customers with higher monthly charges were more likely to churn.
- Low-tenure customers had significantly higher churn rates.
- High-risk customer segments were identified using tenure and monthly charges.

## Dashboard Features
- Executive KPI cards
- Churn distribution analysis
- Contract type analysis
- Payment method analysis
- Monthly charges & tenure analysis
- Risk segmentation dashboard
